1.  用在网盘目录中用初始化一个纯仓库;
    
    git init --bare --shared

2.  在项目目录下将仓库添加进来;（如果你的本地资源没用使用git来管理，你需要git init初始化，并用git commit -a将这些资源提交到本地仓库中）;
    
    git remote add 名字 目录

3.  将代码提交到仓库
    
    git push 名字

4.  ok，现在网盘会自动检测到文件变化同步到远程的网盘中了。
    
    这样，你在家里用网盘同步下仓库，就可以更新出资源了。分支的操作git提供了很强大的支持大家自己研究吧。
